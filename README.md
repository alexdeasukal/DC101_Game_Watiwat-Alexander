# Rock Paper Scissors Game

This is a simple Rock Paper Scissors web game built using **HTML**, **CSS**, and **JavaScript**.  
The player competes against the computer, and the first to reach **5 points** wins the game.

---

## How to Run the Game Using Visual Studio

### Requirements
- Visual Studio Code
- A modern web browser (Chrome, Edge, Firefox)

---

### Steps

1. Open **Visual Studio Code**
2. Click **File â†’ Open Folder**
3. Select the project folder: DC101_Game_WatiwatAlexander
4. Locate the file **index.html**
5. Right-click on `index.html` and choose:
- **Open with Live Server** (recommended), or
- **Open in Default Browser**

---

### Gameplay Instructions

- Click **Rock**, **Paper**, or **Scissors** to make your choice
- The computer will randomly select its move
- Scores update automatically
- The first player to reach **5 points** wins
- Click **Play again** to restart the game
